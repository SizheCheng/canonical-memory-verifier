# Brand assets

The mark shows multiple historical evidence lanes converging on one explicit
current head. It is a compact rendering of the verifier's product behavior, not
a claim that the plugin establishes truth or grants execution authority.

## Files

- `logo.svg`: light listing surface;
- `logo-dark.svg`: dark listing surface;
- `composer-icon.svg`: higher-contrast compact mark.

Every asset is a self-contained 512×512 UTF-8 SVG with no fonts, scripts,
external references, embedded payloads, gradients, or raster content.

## Palette

- Ink: `#17233E`
- Paper: `#F4F1E8`
- Verified head: `#1B8A78`
- Dark-surface verified head: `#5ED6B8`

The assets are covered by the repository's Apache-2.0 license.
