# Support

Canonical Memory Verifier is a local, read-only candidate distributed without
a hosted service or publisher-operated account system.

Support is provided through the public repository's
[GitHub Issues page](https://github.com/SizheCheng/canonical-memory-verifier/issues).
Do not include memory source bytes, secrets, private paths, access tokens,
personal data, or proprietary bundles in an issue. Reproduce problems with the
bundled synthetic fixtures or a newly created minimal synthetic bundle.

Security-sensitive reports should follow [SECURITY.md](SECURITY.md).
